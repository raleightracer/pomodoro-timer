# Simple Pomodoro Timer

A lightweight, distraction-free Pomodoro timer designed to help focus on work and study sessions. Built as a hands-on project to practice clean UI design and precise time management logic.

## Features

*   **Classic Pomodoro Structure:** 25-minute work sessions followed by 5-minute short breaks.
*   **Intuitive Controls:** Simple Start, Pause, and Reset functionality.
*   **Responsive Design:** Clean layout that looks great on desktop

## Built With

*   [HTML5 / CSS3 / JavaScript]
